Pseudocódigo: A Ponte Entre a Lógica e o Código

Até agora, planeámos os nossos algoritmos usando a linguagem do dia a dia. Isso é ótimo para a lógica, mas os computadores precisam de mais estrutura. É aqui que entra o Pseudocódigo (que chamaremos de Portugol).

O Portugol é uma linguagem intermediária. Não é tão informal como o português, mas também não é tão rígida e cheia de regras como uma linguagem de programação real (como C). Ele permite-nos escrever a nossa lógica de forma estruturada, focando no "o quê" fazer, sem nos preocuparmos com os detalhes técnicos do "como".

Por que usar Portugol?

Ele força-nos a organizar o pensamento em variáveis, comandos de entrada, processamento e saída, tornando a tradução para qualquer linguagem de programação (C, Python, Java, etc.) uma tarefa muito mais simples e direta.

Estrutura e Comandos Básicos do Portugol

Um algoritmo em Portugol tem uma estrutura bem definida:

ALGORITMO "nome_do_algoritmo"
// Seção de Declaração de Variáveis
VAR
  nome_variavel_1 : TIPO_DE_DADO
  nome_variavel_2 : TIPO_DE_DADO

// Seção de Comandos
INICIO
  // Comandos de entrada, processamento e saída
FIM_ALGORITMO

Comandos Essenciais:

  • LEIA(variavel): Lê um valor digitado pelo utilizador e guarda-o na variável especificada.
  • ESCREVA("texto", variavel): Mostra uma mensagem ou o conteúdo de uma variável no ecrã.
  • variavel <- valor_ou_expressao: Atribui um valor (ou o resultado de um cálculo) a uma variável. A seta `<-` significa "recebe".

Comparativo: Somar Dois Números

A melhor forma de entender a transição é ver um exemplo prático. Vamos traduzir um algoritmo que soma dois números de Portugol para C.

1. O Plano (Portugol)

ALGORITMO "SomaDoisNumeros"
VAR
  num1, num2, soma: INTEIRO

INICIO
  ESCREVA("Digite o primeiro número: ")
  LEIA(num1)
  
  ESCREVA("Digite o segundo número: ")
  LEIA(num2)
  
  soma <- num1 + num2
  
  ESCREVA("A soma é: ", soma)
FIM_ALGORITMO

2. A Execução (Linguagem C)

#include 

int main() {
  int num1, num2, soma;

  printf("Digite o primeiro número: ");
  scanf("%d", &num1);
  
  printf("Digite o segundo número: ");
  scanf("%d", &num2);
  
  soma = num1 + num2;
  
  printf("A soma é: %d\n", soma);
  
  return 0;
}

Análise da Tradução

  • A seção VAR em Portugol corresponde à declaração de variáveis em C (ex: int num1;).
  • O comando ESCREVA é traduzido para a função printf() em C.
  • O comando LEIA é traduzido para a função scanf() em C. Note os detalhes técnicos como %d e o &.
  • A atribuição <- torna-se o sinal de igual = em C.
  • Todo programa em C precisa da estrutura #include e da função principal int main() { ... }.

Atividade Prática em Duplas

Agora, em duplas, escolham um dos problemas abaixo. Primeiro, criem a solução em Portugol no papel. Depois, abram um editor de texto e tentem traduzir o vosso algoritmo para a Linguagem C, usando o exemplo da soma como guia.

Problema A: Área do Retângulo

Crie um algoritmo que leia a base e a altura de um retângulo, calcule a sua área (Área = Base * Altura) e mostre o resultado.

Problema B: Antecessor e Sucessor

Crie um algoritmo que leia um número inteiro e mostre na tela o seu antecessor e o seu sucessor.
